# scripts

## `hildaplus-upscaling`

The repository for these scripts is hosted in the group's [gitlab](https://gitlab.imk-ifu.kit.edu/lemg/belda-d/hildaplus-upscaling).

- `hildap_tables.py`: Aggregates smoothed Hilda+ data to a target resolution.
- `launch_tables.py`: Utility to launch `hildap_tables.py` on the Keal cluster and post-process the output.
